tripod.desc=Тренога предназначена для установки приемопередатчика при работе с выносом его из машины. С помощью треноги осуществляются горизонтирование и ориентирование приемопередатчика по азимуту.\nТренога состоит из подпятника, пяты, уровня, диска, гаек, винта и трех ног. Основанием треноги является литой подпятник, в приливах которого крепятся па шарнирах три ноги, стопорящиеся гайками-барашками в любом положении.\nНа шарнирных головках ног нанесены насечки, по которым ноги устанавливаются под одинаковым углом по отношению к подпятнику. Телескопическая конструкция ног позволяет регулировать их длину путем выдвижения труб. Фиксация длины ноги осуществляется винтами. На подпятник установлена пята, которая фиксируется посредством тарелки, поджимаемой гайкой.\nНа пяте установлен диск, который может разворачиваться вкруговую относительно пяты. Стопорение диска относительно пяты производится гайкой. На диске размещены шпонки, которые служат для установки механизма вращения приемопередатчика в определенном положении относительно диска.\nМеханизм вращения приемопередатчика крепится винтом.\nНа шпонке размещен уровень, предназначенный для горизонтирования пяты треноги. На шпонке имеется паз, а с резьбовым отверстием для крепления ориентир-буссоли.\nВ походном положении тренога укладывается в укладку совместно с ориентир-буссолью и оптическим визиром.
